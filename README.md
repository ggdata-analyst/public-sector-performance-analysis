# Public Sector Performance Analysis

## Objective
This project analyzes public-sector performance indicators to demonstrate how data-driven analysis can support evidence-based decision-making in development and government institutions.

## Dataset
The analysis uses publicly available datasets from:
- World Bank Open Data
- UNDP Development Indicators
